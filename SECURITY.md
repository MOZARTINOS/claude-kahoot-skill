# Security Policy

## Reporting a vulnerability

Do not open a public issue for vulnerabilities involving credential handling, authentication flows, unintended account changes, or exposure of private quiz content.

Report security issues privately through GitHub's **Security > Report a vulnerability** feature:

https://github.com/MOZARTINOS/claude-kahoot-skill/security/advisories/new

Include reproduction steps, the affected file or workflow, and the potential impact. Do not include real passwords, cookies, tokens, recovery codes, or private student data.

## Supported version

Only the latest release is supported. Browser automation is version-sensitive; check the review date in `references/browser-automation.md`.
